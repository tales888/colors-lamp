# colors-lamp

# about
This Lab is a basic web application which features a login page and a page where users can add or search their colors.

# technologies
This web application uses HTML, CSS, PHP, JavaScript, and MySQL.

# setup
To set this up to work properly you will need to set up a database and link it in the API files then host the web application.

# access
To run this web application you would host all the files on a web server or localhost then you would be able to access it by going to the specified port.
